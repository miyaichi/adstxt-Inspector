Package Title: AdsTxt Inspector
Package Overview: Ads.txt/App-ads.txt とそれに対応する Sellers.json エントリを検証し分析します。
Description:
プログラマティック広告の透明性とコンプライアンスを確保するための必須ツール。Ads.txt/App-ads.txtとSellers.jsonファイルを自動的に検証・分析し、広告取引の健全性を確保します。

このツールで解決できること

 - Ads.txtファイルの文法エラーや重複エントリを自動検出
 - Sellers.jsonファイルとの照合によるセラーIDの検証
 - パブリッシャーとセラーの関係を可視化
 - コンプライアンス違反のリスクを早期発見

主な機能

✨ リアルタイム検証
 - ウェブサイト上のAds.txtファイルを自動スキャン
 - 文法エラーや重複エントリを即座に検出
 - Sellers.jsonファイルとのクロス検証

📊 詳細な分析
 - 広告取引関係の包括的なサマリー表示
 - DIRECT/RESELLERの関係分類
 - ドメインタイプごとの取引パターン分析

🔍 エラー検出・報告
 - 行番号付きの文法エラーハイライト
 - 必須フィールドの欠落チェック
 - セラーIDの不一致識別
 - 無効な取引関係の警告

💻 使いやすいインターフェース
 - 直感的な概要ダッシュボード
 - エントリーごとの詳細検査
 - リアルタイムのエラーレポート
 - 関係性の視覚的な表示

プライバシーとセキュリティ

 - すべての処理はブラウザ上で完結
 - 外部サーバーへのデータ送信なし
 - 公開されているAds.txt/Sellers.jsonファイルのみを分析
 - トラッキング・アナリティクス収集なし

推奨利用シーン

 - パブリッシャー：自社サイトのAds.txt設定の検証
 - アドテク企業：取引先との関係性の確認
 - 広告担当者：プログラマティック広告の取引透明性チェック
 - 開発者：Ads.txt実装時の動作確認

安全で透明なプログラマティック広告取引のために、ぜひご活用ください。
